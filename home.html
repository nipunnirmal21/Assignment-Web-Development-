Welcome to Our Web Application!

This is a demonstration of a multi-page web application built with HTML, Tailwind CSS, and JavaScript. Explore the different sections using the navigation bar above.

You can register for a new account, log in with existing credentials, view a simulated student profile, and submit a contact form.

Client-side validation is implemented for forms, and you can toggle between light and dark modes.
